## input样式

### 清除默认样式

```css
input {
        border: none;
        outline: none;
        background-color: transparent;
        color: #000000;
        width: 244px;
      }
```

### placeholder字体颜色

input::placeholder

```css

input::placeholder {
        color: #007a9f;
      }
```

### checkbox样式

不能直接使用css，而是使用after遮住原先的checkbox，给after设置样式

```css
 input[type="checkbox"] {
        cursor: pointer;
        position: relative;
        width: 20px;
        height: 20px;
        margin: 2.5px 20px 0px 0px;
        font-size: 14px;
      }

      input[type="checkbox"]::after,
      input[type="checkbox"]:checked::after {
        content: "";
        position: absolute;
        top: -2.5px;
        width: 25px;
        height: 25px;
        border: 1px solid #009cc3;
        border-radius: 7px;
        background-color: #004557;
      }
      input[type="checkbox"]:checked::after {
        content: "✓";
        color: #ffffff;
        display: flex;
        align-items: center;
        justify-content: center;
      }
```
