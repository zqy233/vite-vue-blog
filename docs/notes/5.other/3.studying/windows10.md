## windows cmd指令

```sh
# 盘名：代表切换到相应盘,如以下,切换至e盘
e:   
# 创建文件夹,等价于mkdir
md 文件夹名 
# 切换至该文件夹，输入部分文件名按tab可以自动补全
cd 文件夹名 		 
# 切换到上一级
cd .. 	
# 创建空文件
cd.>文件名.文件后缀 
# vscode打开当前目录
code ./     		 
# 显示当前目录中文件列表
dir 				 
```

## 重复修改文件名后缀

新建bat文件，可将当前文件夹下面所有该后缀名文件改名

示例：

```sh
ren *.txt* *.html*
```

## 使用搜索引擎搜索时去掉一些网站

使用-号来去掉该网站的搜索结果

示例：

```sh
实现自己的脚手架 -csdn-简书-知乎-腾讯云-掘金-博客园-思否
```
