# 实例

## 滚动顶部

`onPageScroll`监听页面滚动

`uni.pageScrollTo`设置滚动到的位置和动画时长

```js
<script>
  export default {
    data() {
      return {
        showBackTop: false
    },
    onPageScroll(e) {
      this.showBackTop = e.scrollTop > 500 ? true : false;
    },
    methods: {
      scrollTop() {
        uni.pageScrollTo({
          scrollTop: 0,
          duration: 100
        })
      }
    }
  }
</script>
```
