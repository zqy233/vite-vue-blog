## 泛型+联合类型+数组

```typescript
function a<T>(arr:T[]|T):T[]|T {
    return arr
}
let b = a([1,2,3])
b=1 //类型正确
b=[1] //类型正确
b=['1'] //ts提示不能将类型“string”分配给类型“number”
```
