# idea使用

## 运行项目

### 默认快捷键

```sh
Alt+Shift+F10
```

### 自定义快捷键

File>Settings>Keymap>搜索run>为run添加快捷键

## 语法补全-main方法

```sh
main
```

## 语法补全-System.out.println();

```sh
sout 
```

## 语法补全-数组for循环

```sh
fori
```

## 安装插件

File>Settings>Plugins>搜索插件

如果加载不出来插件列表，检查Appearance&Behavior>System Settings>HTTP Proxy中是否开启了proxy，如果有则关闭proxy

注意！安装完插件很多时候需要重启idea
